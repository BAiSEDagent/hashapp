# Product Docs

Hashapp is a BYOA money app for agents.

## Core product truth
- User wallet stays in control
- Linked agents get bounded authority
- Hashapp is the control, proof, and receipt layer
- Swap infrastructure is subordinate to the payment story, not the product identity
