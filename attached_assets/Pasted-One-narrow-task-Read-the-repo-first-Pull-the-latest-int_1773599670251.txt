The task
Wire one real Grant Permission call from the UI into the proven Base Sepolia spend-permission path.

Exact contract
Address: 0xf85210B21cC50302F477BA56686d2019dC9b67Ad
Network: Base Sepolia (chainId 84532)
Use the approve(spendPermission) function — not approveWithSignature
Full ABI is in proof/scripts/spend-permission-proof.ts

USDC address
0x036CbD53842c5426634e7929541eC2318f3dCF7e

The SpendPermission struct
struct SpendPermission {
address account;
address spender;
address token;
uint160 allowance;
uint48 period;
uint48 start;
uint48 end;
uint256 salt;
bytes extraData;
}


What to implement
In Activity.tsx, when the user taps Grant Permission on a pending spend permission request:

Use useWriteContract from wagmi
Build the SpendPermission struct with:
account = connected wallet address
spender = a fixed Scout address (use 0xbf8BFDe4B42baa2F4377B8Ebc5D2602d3080a4D4 for now as demo Scout)
token = USDC address above
allowance = permission amount in USDC raw units (multiply by 1_000_000 for 6 decimals)
period = 86400 (1 day)
start = Math.floor(Date.now() / 1000)
end = start + 3600 (1 hour for demo)
salt = BigInt(Date.now())
extraData = "0x"
Call approve(spendPermission) on the contract
On transaction confirmed: call approvePending(id, txHash) with the real tx hash
Show a real Basescan link: https://sepolia.basescan.org/tx/[txHash]
